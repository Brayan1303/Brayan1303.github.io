Elementos Básicos de una Tabla HTML
Elemento Descripción
<table> Define una tabla.
<caption> Título de la tabla.
<thead> Define el encabezado de la tabla.
<tr> Define una fila en la tabla.
<th> Define una celda de encabezado (aparece en negrita).
<tbody> Define el cuerpo principal de la tabla.
<td> Define una celda de datos.
Recuerda usar etiquetas semánticas para una mejor estructura y accesibilidad.
