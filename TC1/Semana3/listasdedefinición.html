Glosario de Términos de Programación

Algoritmo
Un conjunto de instrucciones paso a paso para resolver un problema o realizar una tarea.

Compilador
Un programa que traduce el código fuente escrito en un lenguaje de programación de alto nivel a código de máquina.

Depuración
El proceso de encontrar y corregir errores en el código de un programa.

Variable
Un contenedor para almacenar datos que pueden cambiar durante la ejecución de un programa.

Función
Un bloque de código reutilizable que realiza una tarea específica y puede ser llamado desde otras partes del programa.
